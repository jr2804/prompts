# prompts (and more)

Prompts, instructions, skills, templates, etc. for AI-based coding assistants. Can be used as a source for [saddle-cli](https://saddle.sh/).

## Install

Use `mise` to install the repository. If you don't have `mise` installed, you can install it with:

```bash
curl -fsSL https://mise.sh/install | sh
```

...or on Windows:

```powershell
iwr -useb https://mise.sh/install | iex
```

Run `bunx saddle-cli`, then follow the instructions to install the repository.

## Upcoming skills/ideas

### Anti-AI-Slop

There are several skills already available that are designed to reduce "AI slop" (unnecessary or irrelevant information generated by AI models). However, instead of having multiple may be even overlapping skills, it may be more efficient to have a single "Anti-AI-Slop" skill that can be used across different agents and contexts. Idea: Merge the following skills into one:

- <https://github.com/glaforge/deslopify>
- <https://github.com/rand/cc-polymath/tree/main/skills/anti-slop>
- <https://github.com/alinaqi/claude-bootstrap/tree/main/skills/code-deduplication>

(might be a short skill that has several reference documents)

When the skill is executed, it should contain instructions for the model to identify and remove any unnecessary or irrelevant information from its output, ensuring that the generated content is according to the skill's guidelines.
