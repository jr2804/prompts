# Front Matter

Required metadata keys for project PLAN.md YAML frontmatter:

- `meeting` -- meeting identifier (e.g. `#135-bis-e`)
- `tdoc` -- TDoc number (e.g. `S4-260717`)
- `date` -- meeting date range (e.g. `10 - 17 April 2026`)
- `place` -- meeting location (e.g. `Online`, `Geneva`)
- `source` -- submitting organization (e.g. `HEAD acoustics GmbH`)
- `title` -- contribution title
- `agenda_item` -- agenda item number (e.g. `7.6`)
- `target` -- document target (e.g. `Agreement`, `Discussion`, `Information`)
- `revision_of` -- previous TDoc number if this is a revision; use `~` when not a revision

All keys are required. `PLAN.md` is the single source of truth for metadata.
